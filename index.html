📱 Share App to Employee Android Devices

Send this HTML file directly to any Android phone — no internet needed.

Option 1 – USB Cable (fastest)

Connect phone via USB → copy the weyni.html file to the phone → employee opens it in Chrome browser.

Option 2 – WhatsApp / Telegram

Send the .html file as a document (not image) via WhatsApp or Telegram → employee downloads it → opens in Chrome.

Option 3 – Local WiFi (all devices same time)

Connect all devices to the same router → employees type this computer's IP address in Chrome (e.g. 192.168.1.10) → they see the app live.

⚠ Each device stores its own data. Use Export/Import to sync data between devices.
